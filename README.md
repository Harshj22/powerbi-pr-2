## 🚀 Project Overview
This project demonstrates end-to-end data analysis using Power BI by working with multiple interconnected datasets. The primary focus is on building a robust data model, establishing relationships, and generating meaningful insights through interactive dashboards.

---

## 📁 Dataset Description
The project consists of multiple tables representing a typical business scenario:

- **Sales_Fact.xlsx** → Contains transactional sales data  
- **Returns_Fact.xlsx** → Contains returned orders data  
- **Customer_Dim.xlsx** → Customer details  
- **Product_Dim.xlsx** → Product information  
- **Region_Dim.xlsx** → Regional segmentation  
- **Date_Dim.xlsx** → Date hierarchy for time-based analysis  

---

## 🔗 Data Modeling
- Established relationships between fact and dimension tables
- Implemented **Star Schema**
- Used **Primary & Foreign Keys** for accurate joins
- Ensured data consistency and integrity
